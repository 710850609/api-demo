# api-demo
## 项目简介
- 一个简单增删改查服务
- 前后端分离.前端使用vue,后端基于spring-boot,h2database,mybatis实现


## 编译运行
```bash
git clone https://github.com/710850609/api-demo.git
cd api-demo
mvn clean install
java -jar api-demo-v1.0.jar
```
